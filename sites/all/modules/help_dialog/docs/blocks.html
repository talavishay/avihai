מערכת דרופל מחלקת כל דף לפי איזורים שונים: תוכן, תפריט צדדי וכו'...
למנהל האתר ניתנת  אפשרות  ליצור תיבות של תוכן, לערוך אותן, למחוק אותן ולהחליט באיזה איזור בדף הן יופיעו ובאילו דפים הן יופיעו.
תיבה יכולה להכיל טקסט, תמונה, תפריט ועוד.... למידע נוסף

חיפוש/ניהול תיבות:
structure -> blocks
מערכת ניהול התוכן מרכזת כל התיבות שקיימות באתר ברשימה. התיבות מחולקות לפי האיזורים בדף .

ברשימה זו ניתן לחפש, לערוך, לשנות, למחוק ולהגדיר תיבות.

יצירת/ הוספת תיבה:
כדי להוסיף תיבת תוכן חדשה יש להעזר בתפריט השחור של העורך.
תוכן -> הוסף תוכן -> תיבת מדיה


תהליך הוספת התיבה שונה מתהליך הכנסת התכנים באתר.
ההבדל הוא שבסיום הוספת התוכן יש למקם את התיבה במקום הרצוי ולקבוע את התנאים להופעתה.
את השלב האחרון אנו עושים במסך "ניהול התיבות" שהוזכר בשלב קודם.
במסך זה יש לאתר את התיבה שיצרנו ( היא תופיע בשם הכותרת שמילנו בשלב הראשון).
בדךך כלל היא תופיע בחלק התחתון  ( "מושבת" ) של המסך. שם נמצאות כל התיבות שלא נמצאות בשימוש\מושבתות. ( במידה ואנו עורכים תיבה קיימת היא תופיע ב"אזור" שאליו הוקצתה ובו היא נראת באתר.. לדוגמא "FOOTER FIRST")

לאחר שאיתרנו את התיבה נלחץ על "הגדר" על מנת לכוון את המיקום והתנאים שבו היא תופיע

אם רוצים להגביל את התיבה להופעה בדף ספציפי, יש לרשום את כתובת הדף (לדוגמא: node/11) ולסמן את באופצייה המתאימה ( לדוגמא : "רק בעמודים שברשימה להלן"   )
