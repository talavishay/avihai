תפריט הוא אוסף של קישורים שיכולים להיות פנימיים או חיצוניים. בדרך כלל משמש לניווט.
*באתר זה רק התפריט הראשי (אנגלית ועברית) והפוטר זמין לעורך וניתן לשינוי
** שאר התפריטים (לדוגמא : הצוותים בתפריט האנשים) מנוהלים אוטומטית ע"י המערכת

מערכת ניהול התוכן  מספקת ממשק לניהול תפריטים :
מבנה -> תפריטים -> [ שם התפריט]  (לדוגמא: MAIN-MENU-EN לתפריט הראשי באנגלית)

תפריט ה"פוטר" הוא תפריט דו לשוני . ניתן לתרגם את כותרות הקישורים מתוך ממשק התפריט

מנהל האתר יכול ליצור, לערוך ולמחוק תפריטים. אין הגבלה למספר התפריטים שניתן ליצור. למידע נוסף.
כל תפריט מוצג בתיבה שיכולה להיות מופעלת ואת מיקומה ניתן לקבוע בדף ניהול תיבות.
